รางวัลชนะเลิศเหรียญทอง

การแข่งขันสร้าง Web Application ระดับชั้น ม.4 - ม.6 งานศิลปหัตกรรมครั้งที่ 71 ประจำปีการศึกษา 2566

( 20 มกราคม - 4 กุมภาพันธ์ พ.ศ.2567 )

โดยกรรมการจะสุ่มหัวข้อเว็บไซต์มา 1 หัวข้อจากทั้งหมด 3 หัวข้อ คือ เว็บไซต์ร้านขายของออนไลน์, เว็บไซต์ห้องสมุด, เว็บไซต์ลงทะเบียนเรียน โดยผู้เข้าแข่งขันไม่สามารถรู้หัวข้อได้จนกว่าจะถึงเวลาแข่งขัน โดยหัวข้อที่ได้คือ เว็บไซต์ร้านขายของออนไลน์

ในการแข่งขันครั้งนี้ผมรับหน้าที่เป็นคนเขียนหน้าเว็บ หลังเว็บ และฐานข้อมูลสำหรับเก็บข้อมูลของเว็บไซต์ ครั้งนี้เป็นการแข่งขันครั้งแรกที่เกี่ยวข้องกับการทำเว็บไซต์ ทำให้ผมต้องเตรียมตัวอย่างหนัก และศึกษาด้วยตัวเองเนื่องจากนี่เป็นรายการแรกที่คุณครูส่งไป คุณครูจึงไม่สามารถให้คำแนะนำได้มากนัก โดยในการแข่งขันครั้งนี้ทำให้ผมมีความชำนาญด้านการเขียนเว็บไซต์มากขึ้น ได้เรียนรู้ภาษา php และการจัดเก็บข้อมูลใน MySQL ทั้งหมดนี้ทำให้ผมสนใจการเขียนโปรแกรมและการทำเว็บไซต์ และอยากพัฒนาตัวเองให้เก่งขึ้น

รูปภาพที่เกี่ยวข้อง

* รูปภาพดังกล่าวคือเว็บที่ทำขึ้นมาเพื่อซ้อมสำหรับแข่งจริง เว็บที่ทำขึ้นระหว่างแข่งไม่สามารถนำมาเผยแพร่ได้ *

หน้าล็อกอินสำหรับผู้ใช้และแอดมิน โดยชื่อผู้ใช้และรหัสต้องตรงตามฐานข้อมูลจึงจะสามารถใช้งานได้

หน้าลงทะเบียนสำหรับผู้ใช้ที่ไม่มีบัญชี หากมีบัญชีซ้ำเว็บจะทำการแจ้งเตือน

หน้าโฮมของเว็ปขายของออนไลน์ โดยสามารถค้นหาชื่อ แบรนด์ หมวดหมู่ หรือจะกดปุ่มหมวดหมู่และแบรนด์ เพื่อหาสินค้าตามที่ผู้ใช้ต้องการได้

แอดมินสามารถเช็คหรืออัปเดตสินค้าได้จากหน้านี้

แอดมินสามารถเพิ่มสินค้า แบรนด์ และหมวดหมู่ได้
